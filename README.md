# 4. Niveles de velocidad en EPSON RC+ 7.0

En la pesta√±a **Mover y ense√±ar** puede configurarse el nivel de velocidad, mientras que en **Panel de control** el nivel de potencia.

### Velocidad: ‚ÄúAlta‚Äù vs ‚ÄúBaja‚Äù
- **Velocidad Baja:** el robot se desplaza m√°s lento debido a una limitaci√≥n en el par√°metro *Speed*.  
- **Velocidad Alta:** permite un valor de *Speed* mayor, por lo que el movimiento es m√°s r√°pido.

La velocidad solo determina **qu√© tan r√°pido se mueve el robot**, no la fuerza que aplica.

### Potencia: Power LOW vs Power HIGH

Seg√∫n el manual del Epson T3-401S, este ajuste modifica la **fuerza/torque disponible**, no la velocidad.
- **Power LOW:** el robot trabaja con potencia reducida; limita la fuerza del motor.  
- **Power HIGH:** habilita la potencia completa del motor.

### Relaci√≥n entre velocidad y potencia

- Puede usarse **velocidad alta con potencia baja**, pero el robot podr√≠a no manejar cargas exigentes.
